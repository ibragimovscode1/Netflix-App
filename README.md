# Netflix-App
This is a Cloned version of Netflix App using Swift. 
This is only for personal education purpose.
This clone is featuring most of the features that Netflix has


# Technologies & Features
- CoreData
- URLSession
- Notification Center
- HTTP, RESTful Api
- JSON Parsing
- MVVM Architecture
- Fully Programmatic UI
- AutoLayout
- CoreGraphics
- UIScrollView
- UITableView
- Protocol Delegate
